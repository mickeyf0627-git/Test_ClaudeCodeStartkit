# Aillio 全社共通 指示（Claude Code）

- 顧客の個人情報(PII)・本番の秘密情報を、プロンプト / ツール入力 / コードに含めない。開発は合成・マスキングデータで行う。
- 本番環境（本番DB・本番システム）に対して操作しない。
- 生成コードは人間レビューを経てマージする。
- 未承認のMCPサーバーを追加・接続しない（申請制）。
- 秘密情報ファイル（.env / secrets / 鍵 / 認証情報）を読み書きしない。

（プロジェクト固有の規約は各リポジトリの CLAUDE.md / `.claude/rules` に追記する。本ファイルは全社共通の最低ライン。）
