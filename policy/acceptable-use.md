# Claude Code 利用規程（最小版）

対象：社内で Claude Code を業務利用する全員。利用開始時に同意すること。

## A. アカウント・環境
1. **個人アカウント利用禁止** — 業務は会社Orgアカウントのみで利用する。
2. **設定の改ざん・回避禁止** — managed設定の無効化や、第三者プロバイダ（`ANTHROPIC_BASE_URL` 等）への切替による統制回避を行わない。

## C. MCP・拡張
3. **MCPは承認制** — 未承認MCPの追加・接続を行わない。利用時は 申請 → `mcp-security-review`（一次審査）→ 人間承認 → Owner が `allowedMcpServers` 登録 の手順を経る。

## 補足（規程本文外・データ取扱の注意）
- 顧客PII・本番の秘密情報をプロンプト/ツール入力に入れないこと（データ境界の徹底。起動時にもリマインドが表示される）。
- インシデント・利用状況の監視は別途の監査（OpenTelemetry）設計で扱う。
